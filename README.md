# Analog Electronics Laboratory Works

This repository documents my hands-on experience with fundamental electronic components and circuit design.

## Experiment: Bipolar Transistor Basics
* **Objective:** Examining BJT operating principles, establishing stable biasing, and analyzing amplifier performance.
* **Key Analyses:** * Measured and verified device currents ($I_B, I_C, I_E$) and terminal voltages.
    * Evaluated transistor parameters ($\alpha, \beta$) for device characterization.
    * Observed AC voltage swing limits and signal clipping to define the linear operating region of amplifiers.
* **Tools Used:** Oscilloscope, DC Power Supply, Function Generator, Multimeter.
